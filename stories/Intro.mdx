# Select

Элемент, который позволяет выбирать данные из выпадающего списка опций. По умолчанию список опций
скрыт и он появляется при взаимодействии пользователя с Select.

При необходимости ознакомьтесь с [исходным кодом](https://github.com/2e32/react-select) компонента.

## Начните работать

В storybook представлены материалы, позволяющие ознакомиться с функционалом селекта.

_Свойства (props)_:

- изучите список свойств, которые принимает компонент
- изменяйте значения свойств, чтобы увидеть какой эффект они дадут
